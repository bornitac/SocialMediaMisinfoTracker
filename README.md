📘 SMM (Social Media Misinformation Tracker)
This is an interactive Shiny App that exlores the trends of various misinformation topics across
the United States using public Google Trends data. The users are able to pick between some well-known 
misinformation themes - like election fraud 🗳️, and climate change hoaxes🔥 - and instantly see how 
interest in these conspiracies has changed over the past 5 years. 
This app includes:
- A time-series plot📈 which shows the trends in spikes
- A state-level map 🗺️ which shows differences in searches across states
